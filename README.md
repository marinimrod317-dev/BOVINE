# Nimo Pics - Photography Website

## Description
Nimo Pics is a professional photography and artistic work showcase website. It features a collection of 18 artistic projects created by a team of expert photographers and artists.

## Features
- **Home Page** - Landing page with introduction to Nimo Pics
- **Highlights Gallery** - Showcases 18 artistic photography projects
- **Reviews** - Section for customer reviews and testimonials
- **User Authentication** - Login and registration system for users
  - Login page for existing users
  - Registration form for new user accounts

## File Structure
```
Autonomous/
├── Home.html           # Main landing page
├── highlights.html     # Gallery of 18 artistic projects
├── Reviews.html        # Reviews and testimonials section
├── login.html          # User login page
├── Register.html       # User registration form
├── project 1-18.jpeg   # High-quality photography images
└── README.md           # Project documentation
```

## How to Use
1. Open `Home.html` in a web browser to view the main website
2. Navigate to the Highlights page to view the gallery of artistic work
3. Check the Reviews section for customer testimonials
4. Create an account using the Registration form
5. Login with your credentials

## Technologies Used
- HTML5
- CSS3
- JavaScript (for form handling and interactivity)
